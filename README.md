# Student Registration System

A simple full-stack web application for registering students and storing their details in a MySQL database.

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MySQL

## Why These Technologies?

This stack was chosen because it is lightweight, easy to understand, and allows both frontend and backend development using JavaScript. MySQL provides reliable storage for structured student data.

## Features

* Register a student
* Store student records in MySQL
* Retrieve all students using REST APIs

## REST APIs

* **POST /students** – Add a new student
* **GET /students** – Get all student records

## Database

Database: `studentdb`

Table: `students`

Fields:

* id
* name
* enrollment_number
* email
* mobile_number
* branch
